# ✅ Task Buddy (React To-Do App)

## 📌 Project Description
Task Buddy is a simple and interactive task management application built using React. Users can add, delete, and manage their daily tasks efficiently.

##  Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Real-time UI updates
- Responsive design

##  Technologies Used
- React.js
- JavaScript (ES6)
- HTML
- CSS

##  Project Structure
- src/
  - components/
  - App.js
  - index.js
- public/
  - index.html
